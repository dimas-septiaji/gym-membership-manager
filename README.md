# Gym Membership Manager

A Flutter-based gym membership management application integrated with PHP REST API and MySQL database.

## Features
- Member management
- Membership package management
- Membership expiration tracking
- Product inventory
- Sales transactions
- Invoice history

## Tech Stack
- Flutter
- Dart
- PHP REST API
- MySQL
